WEB интерфейс

Проект WEB интерфейса для CBSD называется ClonOS и реазлизован в виде отдельного дистрибутива

Сайт проекта: clonos.tekroutine.com

Также, с получением определенного количества комментариев о работе ClonOS от пользователей, он будет опубликован в портах FreeBSD

На данный момент это не production-ready продукт, тем не менее, часть функционала работает стабильно, например:

  • WEB интерфейс для создания и управления jail
  • WEB интерфейс для bhyve
  • WEB интерфейс для работы с формами (хелперами) CBSD
  • WEB интерфейс для работы с исходными кодами и базой FreeBSD

Разработчики очень заинтересованы в тестировании и получения обратной связи о проекте
