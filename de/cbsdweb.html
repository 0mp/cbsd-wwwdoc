WEB interface

WEB interface project for CBSD called ClonOS and implemented as a separate FreeBSD distribution

Site of the project: clonos.tekroutine.com

Also, when we obtain a certain number of comments about the ClonOS from users, it will be published in the FreeBSD ports

At present it is not production-ready product, however, the part of functional is stable, e.g.:

  • WEB interface to create and manage jail
  • WEB interface for bhyve
  • WEB interface for CBSD forms (helpers)
  • WEB interface to work with FreeBSD sources and bases

The developers are very interested in testing and feedback
